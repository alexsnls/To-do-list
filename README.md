# To-do-list

## Tecnologias Utilizadas

Backend:
Node.js
Express
TypeScript
PostgreSQL 
API RESTful
Camadas de serviço e repositório

Frontend:
React
Vite
TypeScript
CSS

## Como Rodar o Projeto
### Clone este repositorio https://github.com/alexsnls/To-do-list.git
rode no terminal:
cd backend
npm install
npm run dev

Ja para rodar o frontend
cd frontend
npm install
npm run dev

## Variáveis de Ambiente
Crie um arquivo .env na pasta code-api com as seguintes variáveis:
env
Copiar
Editar
DB_HOST=localhost
DB_PORT=5432
DB_USER=seu_usuario
DB_PASSWORD=sua_senha
DB_NAME=todo_list_db
